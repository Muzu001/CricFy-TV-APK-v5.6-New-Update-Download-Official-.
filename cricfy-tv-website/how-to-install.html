Installing on Smart TV

Allow installation from unknown sources on your Smart TV.

3

  3. Select and open the app
  4. Use remote to navigate
  5. Start streaming on big screen

Troubleshooting Common Issues

Installation Failed or App Won't Install

  • Ensure you have enough storage space (at least 50MB free)
  • Check that Unknown Sources is enabled
  • Try restarting your device and installing again
  • Clear cache of your file manager app

App Crashes on Startup

  • Restart your device completely
  • Clear app cache and data
  • Ensure you have a stable internet connection
  • Try reinstalling the app

Streaming Issues or Buffering

  • Check your internet connection speed
  • Try switching to a different server in the app
  • Close other apps using internet
  • Restart your router if needed

Can't Find Downloaded APK File

  • Check your Downloads folder
  • Look in your browser's download history
  • Use a file manager app to search for "cricfy"
  • Re-download if the file is missing
